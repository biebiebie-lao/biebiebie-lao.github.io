## 环境准备

首先，我们需要准备 ESP32 的开发环境。本文以 Windows 系统为例，介绍 ESP32 开发环境的配置过程。

## 安装 Arduino IDE

推荐使用 Arduino IDE 来开发 ESP32，下载安装完成后，需要添加 ESP32 开发板支持。

## 配置开发板

在 Arduino IDE 中，通过 `文件` → `首选项` → `附加开发板管理器网址` 添加 ESP32 开发板链接。

## 第一个程序：Blink

让我们来写第一个 ESP32 程序，让开发板上的 LED 灯闪烁。

```cpp
void setup() {
  pinMode(2, OUTPUT);
}

void loop() {
  digitalWrite(2, HIGH);
  delay(1000);
  digitalWrite(2, LOW);
  delay(1000);
}
```

将代码上传到 ESP32，就可以看到 LED 灯在闪烁了！
