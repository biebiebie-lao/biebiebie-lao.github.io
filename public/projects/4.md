# 反应训练灯

## 项目简介

基于 ESP32-C6 的分布式反应时间测量系统，通过无线方式实现多从机协作，支持距离传感器和震动传感器两种触发方式，测量精度达微秒级。

配合移动 App 上位机，可实现反应速度训练、数据记录与分析。

## 系统架构

```
┌─────────────┐     BLE UART      ┌─────────────┐    ESP-NOW    ┌─────────────┐
│  移动 App   │ ◄───────────────► │    主机    │ ◄──────────► │    从机     │
│ (Quasar)   │                    │  (ESP32-C6)│              │  (ESP32-C6) │
└─────────────┘                    └─────────────┘              └─────────────┘
```

## 硬件清单

### 主机
- ESP32-C6 开发板
- BLE 天线

### 从机
- ESP32-C6 开发板
- 距离传感器（VL53L0X）或震动传感器
- RGB LED

## 功能特性

### 测量模式
- **距离模式**：根据预设距离（30cm/40cm/50cm/60cm）计算声音传播延迟
- **震动模式**：基于震动触发计时

### 系统特性
- **主从式架构**：一主机多从机，支持多设备协同测量
- **无线通信**：ESP-NOW 实现主机与从机间的低延迟通信
- **微秒级精度**：配合 `esp_time_sync` 时间同步，测量精度极高
- **实时反馈**：LED 颜色变化实时显示状态

### 上位机功能
- 蓝牙 BLE 连接 ESP32 设备
- 标准模式：预设反应距离（30cm、40cm、50cm、60cm）
- 自定义模式：自定义反应距离（1-100cm）
- 数据记录：自动保存训练记录，查看历史数据

## 技术栈

### 下位机
- **芯片**：ESP32-C6
- **开发框架**：ESP-IDF 5.3.x
- **无线协议**：ESP-NOW
- **通信协议**：BLE UART 透传

### 上位机
- **框架**：Vue 3 + Quasar
- **移动端**：Capacitor
- **蓝牙通信**：@capacitor-community/bluetooth-le
- **状态管理**：Pinia

```
src/

├── components/     # 公共组件

├── constants/      # 协议常量

```

## 快速开始

### 下位机编译
```bash
cd reaction_-now
idf.py set-target esp32c6
idf.py build
idf.py -p COM13 flash
idf.py -p COM13 monitor
```

### 上位机开发
```bash
cd now-reaction
yarn
quasar dev
```
