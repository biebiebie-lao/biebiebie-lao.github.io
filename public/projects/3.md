# ESP32 物联网网关

## 项目简介

支持多种传感器接入的物联网网关，实现数据聚合与上传。

## 系统架构

```
传感器节点 → ESP32 网关 → MQTT Broker → 数据可视化
```

## 支持的协议

- MQTT (主要)
- HTTP RESTful
- WebSocket
- Modbus RTU

## 硬件连接

| GPIO | 功能 |
|------|------|
| GPIO21 | I2C SDA |
| GPIO22 | I2C SCL |
| GPIO4 | OneWire 总线 |
| GPIO34 | ADC 输入 |

## 配置方式

通过 Web 界面配置 WiFi、MQTT 服务器、传感器参数。

```json
{
  "wifi": {
    "ssid": "YourWiFi",
    "password": "YourPassword"
  },
  "mqtt": {
    "server": "broker.hivemq.com",
    "port": 1883,
    "topic": "iot/gateway/data"
  }
}
```

## 集成 Node-RED

网关数据可以直接上报到 Node-RED 进行流程编排和数据可视化。

## 应用场景

- 智能温室监测
- 工业设备监控
- 仓库环境管理
