# 智能温湿度监控系统

## 项目简介

基于 ESP32 和 DHT11 传感器实现的温湿度监测系统，支持数据上报和远程查看。

## 硬件清单

- ESP32 开发板
- DHT11 温湿度传感器
- 杜邦线若干
- USB 数据线

## 功能特性

- 实时监测环境温度和湿度
- 通过 WiFi 上报数据到服务器
- 支持 HTTP/MQTT 协议
- 低功耗设计

## 使用方法

```cpp
#include <WiFi.h>
#include <DHT.h>

#define DHTPIN 4
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();
  // 连接 WiFi...
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();
  // 上报数据...
}
```
