# 强化学习机械臂控制

## 项目简介

使用深度强化学习算法控制机械臂完成抓取任务。

## 技术栈

- Python 3.8+
- PyTorch 1.9+
- OpenAI Gym
- ROS (可选)

## 算法实现

采用 DDPG (Deep Deterministic Policy Gradient) 算法：

### 状态空间

- 机械臂关节角度
- 末端执行器位置
- 目标物体位置

### 动作空间

- 关节角速度
- 夹爪开合程度

### 奖励设计

- 接近目标时正奖励
- 成功抓取时大奖励
- 碰撞或超时负奖励

## 训练结果

经过 10000 回合训练后，成功率达到 85% 以上。
